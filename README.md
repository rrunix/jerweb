# Educational Mini-Games and Visualizations

Interactive visualizations for teaching networking and computer science concepts.

## Games & Visualizations

- **Router Game** - Simulate routing protocols and packet forwarding
- **CIDR & Longest Prefix Matching** - Understand subnet masking and routing decisions
- **Static TCP Bandwidth Distribution** - Visualize bandwidth sharing among TCP connections
- **Dynamic TCP Bandwidth Distribution** - Interactive bandwidth allocation simulation

## Running Locally

This is a Jekyll site. To run locally:

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000` in your browser.
