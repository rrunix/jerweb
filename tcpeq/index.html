📊 ¿Qué muestra esta gráfica?
  • TCP es "fair" (equitativo): Cuando múltiples conexiones TCP comparten un mismo enlace, el protocolo TCP distribuye el ancho de banda de manera equitativa entre todas ellas.
  • División equitativa: Si hay N conexiones compartiendo un enlace con capacidad R, cada conexión obtiene aproximadamente R/N del ancho de banda total.
  • Ejemplo práctico: Con 4 conexiones compartiendo 100 Mbps, cada una obtiene ~25 Mbps.
  • Mecanismo: Esto se logra mediante los algoritmos de control de congestión de TCP (como AIMD - Additive Increase Multiplicative Decrease).
